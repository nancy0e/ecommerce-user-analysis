
ecommerce-user-analysis（电商用户画像分析项目）

本项目使用 Python 对电商消费者行为数据进行分析，包含：

KMeans 行为聚类

RFM 用户价值模型

多维用户标签体系构建

雷达图 / 热力图 / 旭日图 / 树状图 可视化

用户画像分析报告

适合：

数据分析作品集

简历项目

数据挖掘课程作业

电商运营/产品用户洞察

📊 分析流程（Analysis Workflow）
1️⃣ 数据探查（Data Exploration）

包括：

数据基本信息（行数/列数）

缺失值、异常值

特征类型识别

金额字段清洗

日期字段解析

2️⃣ KMeans 行为聚类（User Behavior Clustering）

特征工程包含：

Monetary（消费金额）

Frequency（购买频率）

Loyalty_Score（品牌忠诚度 + 满意度）

Engagement（优惠使用 + 会员）

研究时间、退货率、评分等行为变量

分类特征 One-Hot 编码

数值特征标准化

使用 Numpy 手写 KMeans（因为用户本地 sklearn 安装失败）。

输出：

5 类行为标签（深度研究型、高参与型、价格敏感型、年轻高潜、流失倾向）

雷达图展示特征差异

3️⃣ RFM 用户价值分层（Value Segmentation）

定义：

R：距最近购买天数

F：购买频率

M：消费金额

五分位数评分 → RFM_total → 分层：

RFM_total	价值等级
13–15	VIP 超高价值
10–12	高价值
7–9	中价值
4–6	低价值
3	潜在流失

输出：

RFM 热力图

RFM 雷达图

用户价值分层表

4️⃣ 多维用户标签体系（行为 × 价值）

将：

行为聚类（5 类）

RFM 价值等级（5 类）

组合成：

25 种复合标签，如：
“VIP · 深度研究型用户”
“中价值 · 价格敏感型用户”

可视化：

旭日图（按人数）

树状图（按金额）
