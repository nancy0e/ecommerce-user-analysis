 📊 电商用户画像与用户价值分析报告  
（Ecommerce User Segmentation & Value Analysis Report）

---

 1. 项目背景

本项目基于电商消费者行为数据，通过以下方法构建完整的用户画像体系：

- **KMeans 行为聚类模型**
- **RFM 用户价值模型**
- **行为 × 价值 的多维标签体系**
- **可视化分析（雷达图、热力图、旭日图、树状图）**

目标是理解不同消费者群体的行为特点与价值差异，为企业运营提供有针对性的用户策略。

---

 2. 数据概览

- 数据量：**1000 条记录，28 个字段**
- 包含用户属性、购买行为、消费金额、参与度、满意度等信息
- 关键字段示例：Age、Frequency、Brand_Loyalty、Purchase_Amount、Time_of_Purchase 等

数据清洗工作包括：

- 金额字段去除 `$` → 转为 float
- 日期解析为 datetime
- One-Hot 编码分类字段
- 数值标准化处理

---

 3. KMeans 行为聚类

 3.1 特征工程

构建行为类变量（部分）：

| 特征名 | 含义 |
|--------|------|
| Monetary | 本次消费金额 |
| Frequency | 购买频率 |
| Loyalty_Score | 品牌忠诚度 + 满意度 |
| Engagement | 是否参与优惠 + 是否会员 |
| Product_Rating | 产品评分 |
| Return_Rate | 退货率 |
| Time_Spent_on_Research | 决策前研究时间 |
| Time_to_Decision | 决策时长 |

分类变量经过 One-Hot 编码后进入模型。

---

 3.2 聚类结果总结

最终得到 **5 类用户行为分群**：

- **Cluster 0：深度研究型用户**  
  研究时间长、决策慢、信息敏感度高  
- **Cluster 1：高参与型用户**  
  积极使用优惠、活动参与度高  
- **Cluster 2：价格敏感型用户**  
  Monetary 低，折扣敏感度高  
- **Cluster 3：潜在流失型用户**  
  满意度与活跃度偏低  
- **Cluster 4：年轻高潜用户**  
  年龄低、成长潜力高

 4. RFM 用户价值模型

 4.1 指标定义

- **R（Recency）**：距最近一次购买的天数  
- **F（Frequency）**：累计购买频率  
- **M（Monetary）**：累计消费金额  

R、F、M 各维度按五分位打分（1~5）。

---

 4.2 用户价值分层结果

| RFM 总分 | 价值区间 |
|----------|----------|
| 13–15 | VIP 超高价值用户 |
| 10–12 | 高价值用户 |
| 7–9 | 中价值用户 |
| 4–6 | 低价值用户 |
| 3 | 潜在流失用户 |

 分层统计如下：

- 中价值用户：458  
- 高价值用户：304  
- 低价值用户：159  
- VIP 超高价值：71  
- 潜在流失：8

 5. 多维用户标签体系（行为 × 价值）

将：

- **行为聚类（5 类）**
- **价值模型（5 类）**

结合生成 **25 种最终用户标签**：
如：

- VIP · 深度研究型用户  
- 高价值 · 潜在流失行为型用户  
- 中价值 · 价格敏感型用户  

输出文件：

 7. 用户洞察总结（Insights）

以下洞察结合 **KMeans 行为聚类、RFM 用户价值模型，以及最终的行为 × 价值多维用户标签体系** 得出。  
每个洞察都基于数据特征（均值、得分、分布），并提供可落地的业务策略。

---

 ⭐ 7.1 VIP 高价值用户  
**来源模型：RFM（RFM_total 13–15） + 部分聚类标签**

 📌 数据证据（How we know）
- Monetary：显著处于最高分位  
- Frequency：多次购买  
- Recency：近期仍有消费  
- Loyalty_Score：偏高（来自聚类特征）  
- 某些 VIP 用户主要分布在 **深度研究型 + 高参与型** 聚类中

 📌 用户画像
- 消费能力强  
- 购物稳定且忠诚  
- 愿意花时间选好商品  
- 评价、评分偏高，对品牌信任度强

 📌 运营策略（可落地）
- 会员专属权益：专属折扣 / 提前购 / 积分翻倍  
- 定制化推荐：高客单品 / 新品优先  
- 建立强关系链：客服优先级、生日礼、线下活动

---

 ⭐ 7.2 高价值用户（非 VIP）  
**来源模型：RFM（RFM_total 10–12）**

 📌 数据证据
- Monetary：“稳定高”，但不如 VIP 极端  
- Frequency：高于人群均值  
- Recency：较好，未出现明显流失迹象  
- 这些用户多数落在**高参与型 & 年轻高潜型**行为聚类中

 📌 用户画像
- 消费稳定的中坚力量  
- 具备高成长性  
- 对活动、优惠接受度较高  
- 兼具理性与冲动消费特征

 📌 运营策略
- 强化复购：满减券、凑单推荐  
- 推动客单价提升：相关品类捆绑推荐  
- 打造成长路线：积分体系升级激励

---

 ⭐ 7.3 中价值用户（规模最大）  
**来源模型：RFM（RFM_total 7–9）**

 📌 数据证据
- 人数最多（占比最高）  
- Monetary、Frequency 都在中位区间  
- 分布在多个聚类：价格敏感型 / 深度研究型 / 年轻高潜型

 📌 用户画像
- 购买习惯稳定  
- 消费金额不高但具成长空间  
- 可能是未来“高价值用户”的候选群体

 📌 运营策略
- 精准推荐促进二次购买  
- 针对兴趣品类推送新品  
- 培养用户忠诚度（积分、签到、互动任务）

---

 ⭐ 7.4 价格敏感型用户  
**来源模型：KMeans 聚类（Cluster 2）**

 📌 数据证据
- Monetary：低  
- Engagement：折扣使用率高  
- Frequency：低–中，但对价格波动敏感  
- Social Media Influence：偏高（多数来自促销渠道）

 📌 用户画像
- 对价格高度敏感  
- 非必需品购买意愿低  
- 促销期间更活跃

 📌 运营策略
- 满减券、限时降价  
- 大促活动重点触达  
- 强调价值感（如：对比、评价、性价比标签）

---

 ⭐ 7.5 深度研究型用户  
**来源模型：KMeans 聚类（Cluster 0）**

 📌 数据证据
- Time_Spent_on_Product_Research：显著最高  
- Time_to_Decision：决策慢  
- Product_Rating：倾向给出较高评分  
- 研究后转换率较高（来自购买行为）

 📌 用户画像
- 理性、信息敏感度高  
- 做足研究才下单  
- 受商品内容与评价影响明显

 📌 运营策略
- 提供专业内容：指南、评测、对比图  
- 优化商品详情页面（参数、材质、场景图）  
- 增强“专家推荐”或“用户评价质量”

---

 ⭐ 7.6 高参与型用户  
**来源模型：KMeans（Cluster 1）**

 📌 数据证据
- Engagement 分数最高（折扣使用 + 会员）  
- Frequency 偏高  
- Social Media 互动较多

 📌 用户画像
- 活跃用户，喜欢参与活动  
- 熟悉平台玩法  
- 对优惠机制敏感

 📌 运营策略
- 打造常规活动的“核心人群”  
- 互动任务（签到、抽奖、积分任务）  
- 会员体系绑定：权益强化、兑换商城

---

 ⭐ 7.7 潜在流失用户  
**来源模型：RFM（RFM_total = 3） + KMeans（Cluster 3）**

 📌 数据证据
- Recency：极高（长时间未购买）  
- Frequency、Monetary 均处于低分位  
- Customer_Satisfaction 低于平均水平  
- 多位用户聚集在“潜在流失行为型”聚类中

 📌 用户画像
- 已停止活跃  
- 对产品不满意或对比竞争对手  
- 可能因价格/体验/需求变化而流失

 📌 运营策略
- 唤醒营销：短信提醒 → 满减优惠  
- 回访调查：收集流失原因  
- 强调利益点：免运费券 + 新人折扣级别优惠  
- 若挽回失败，可移入“冷启动再激活”池
